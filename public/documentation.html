GRD API supports few queries

Limit:

By default GRD will respond with 1 item. With limit query you can increase amount of items:

/api/q?limit=25
Country:

By default GRD will respond with 1 random country. With country query you can default to selected country (if selected country is not on the list it will be replaced with a random one):

/api/q?country=poland
Friends:

Friends query only works with limit query. By default it will add from 0 to 3 friends per person. You can increase the limit to up to 10:

/api/q?friends=8
Chaining:

You can chain queries together by seperating them with & symbol.

/api/q?friends=8&limit=25
